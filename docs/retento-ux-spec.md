# Retento UX仕様書

Retentoは「最上級のUX」を目指す英単語学習アプリであり、本仕様書はその核となるデザイン思想・インターフェース・AI活用・技術アーキテクチャを網羅する。

## 1. UXの5大柱
1. 摩擦ゼロの学習フロー
2. 高品質なAIコーチング
3. 科学的根拠とAIの融合
4. 持続的モチベーションの設計
5. 待ち時間ゼロを支える非同期アーキテクチャ

## 1.5. デザインルール：Light-First Theme
- 背景面はすべてライト（白〜非常に淡いグレー）で統一し、質感の濃いカード枠は極力排除。
- テキストは高いコントラスト（黒〜濃グレー）で統一し、重要な説明は適度な字間・行間で余裕を持って表示。大見出しはしっかりと余白を取り、視認性を高める。
- セクションや一覧は余白とグリッドだけで区切りをつけ、境界線やシャドウはごくごく薄く、囲み感は与えない。
- アクセントカラー `#c2255d` は下線・ボタンなどに限定し、背景全体には使わない。
- 写真やモノトーンのグラフィック（hero の photo/illustrationなど）は詩的な文脈を添えるために使い、濃いフィルタやダークグラデーションは避ける。

## 2.0. プロフェッショナルなデザイントーン
- テキストで「科学」や「データ」の語彙を意識し、直感的なAI賛美ではなく、説明責任と信頼性を軸に据える。
- 短いキャプション（例：「傾向分析」「非同期生成」「信頼性」）をセクション内で繰り返し使い、データ主導の文脈を強める。
- 広告的なアニメーションや派手なグローは避け、レイヤー構造と整然とした余白で静かなプロフェッショナルさを伝える。
- Hero/Sectionの本文はモダンなグリッドに沿って横幅を絞り、余裕のある読み味を優先する。

## 2. 第一の柱：摩擦ゼロの学習フロー
### デザインコンセプト
- モダンかつミニマルなライトテーマ。
- モノトーンをベースにアクセントカラー#B8084Fを限定使用（下線、プログレスバー、ホバー/タップエフェクト等）。

### 学習の開始と再開
- アプリ起動直後に問題画面を表示し、ホーム・メニューは介さない。
- 中断した学習は確認ダイアログなしでそのまま再開し、再開時の摩擦を排除。

### 問題画面とフィードバック
- 「下線部の日本語訳」を問う形式。文脈を保つ例文と、アクセントカラーで下線を引いたキー単語。
- カード型UI。選択肢タップでボタン背景が薄い緑／赤に静かに変化し、派手なアニメーションは使わない。
- 不正解時には選択肢が赤、正解が緑で即時提示。

### レスポンシブレイアウト
- スマホ縦画面：回答後に上下分割。上部に問題カード、下部にAI解説カード。
- PC/タブレット横画面：左右分割。左に問題カード、右にAI解説。

### スワイプ操作と遷移
- AI解説読了後、問題カードのスワイプで自信度入力（上＝マスター、右＝通常復習、左＝短期復習）。
- 物理的フィードバックとしてカードの傾きや薄文字表示。
- 不正解の問題は自動的に短期復習に組み込まれ、スワイプ不要で次へ移行。
- 次の問題は常にプリロードされており、スワイプ完了と同時に新しいカードが表示される。

## 3. 第二の柱：高品質なAIコーチング
### 個別フィードバック
- 事前生成されたAI解説を提示。語源、類義語比較、記憶術（ニーモニクス）の3アプローチでAha体験。

### セッション全体のテーマ解説
- 10問セッションは動的テーマ（語源、類義語、カテゴリ）を中心に組成。
- セッション終了後に出題意図を明かす総合フィードバックで、知識の「面での整理」を実感させる。

### 能動的テーマリクエスト
- 1日3回まで自由にテーマをAIにリクエスト可能。
- 非同期キューで処理し、完了通知はプッシュ。生成待ちなし。

## 4. 第三の柱：科学的根拠とAIの融合
### ハイブリッド・テーマ決定アルゴリズム
1. 忘却曲線に基づく「今日復習すべき単語リスト」を抽出。
2. AIがリスト内で関連性の高い語を発見しグルーピング。
3. 10問のセッション構築：テーマを核に新規単語、復習必須単語、レベルに合わせた新語で構成。
4. 難易度フィルターでユーザーとの乖離を見張り、最適な「在庫」セッションを準備。
5. フィードバックをテーマ性・復習性で分岐。

### 復習間隔の最適化
- 自信度スワイプに加え、回答速度も加味して次回復習の間隔を調整。

## 5. 第四の柱：持続的モチベーション
### ランクシステム（絶対評価）
- S+, S, S-, A+, A… といった等級体系で累積スコアによって次のランクまでの進捗を可視化。

### ランキング（相対評価）
- オプトイン式。デフォルト非表示で、「ランキングに参加しますか？」画面で明示的に参加を選択した場合のみ機能。
- 総合ランキング：累積スコアを比較。
- 週間ランキング：週間学習時間・連続学習日数など「努力量」を指標にする。

### 初期レベル診断テスト
- 本番と同じカードUI/スワイプ/静的フィードバックで50問のCAT式テスト。
- テスト中に自然に操作方法を習得させる。

## 6. 第五の柱：ゼロ待ち時間を支えるアーキテクチャ
### AI個別フィードバックの事前生成
- 単語と出会う前に語源・解説を生成・保存。ユーザーは即取得できるため待ち時間ゼロ。

### AI総合フィードバックの非同期生成
- 10問目回答時をトリガーに生成開始。ユーザーが解説を読み終える数秒間で完了。

### セッション生成の非同期キュー
- セッション完了ごとに「次のセッション生成」ジョブをキューに投入。
- 空いているワーカーが順に処理し、負荷を分散。
- ユーザーは常に生成済みセッションを利用可能（例: 1日30セッション上限）で待ち時間なし。

この仕様により、Retentoは科学的理論・高品質なAI・洗練されたUXを組み合わせた学習体験を提供する。
